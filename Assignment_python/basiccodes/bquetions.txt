4. What is the difference between a list and a tuple?
Answer:

List: Can change (mutable) → []

Tuple: Can’t change (immutable) → ()

Example:

my_list = [1, 2, 3]
my_tuple = (1, 2, 3)

6. What is the difference between is and ==?
Answer:

== checks values

is checks memory location

Example:


a = [1, 2]
b = [1, 2]
print(a == b)  # True
print(a is b)  # False

11. What is a loop? Difference between for and while loop?
Answer:
Loop runs the same code multiple times.

for → used when we know how many times to run

while → runs until condition becomes false

for i in range(3):
    print("Hi")

i = 0
while i < 3:
    print("Hi")
    i += 1

13. What is the use of break, continue, pass?
Answer:

break: Stops the loop

continue: Skips current loop step

pass: Does nothing (used as placeholder)

14. What are arguments and parameters in a function?
Answer:

Parameters are the names in the function definition.

Arguments are the actual values we pass.


def greet(name):  # name is parameter
    print("Hi", name)

greet("Alice")  # "Alice" is argument

15. What is exception handling?
Answer:
It prevents the program from crashing when an error happens.


try:
    x = 1 / 0
except ZeroDivisionError:
    print("Can't divide by zero")
